# Deep Learning Bootcamp (Le Wagon)

Este repositório contém meus estudos e projetos desenvolvidos durante o bootcamp de Deep Learning da [Le Wagon](https://www.lewagon.com/).

## Estrutura
- `01_fundamentals`: conceitos básicos de DL
- `02_training`: processo de treinamento de modelos
- `03_cnn`: utilizando CNNs
- `04_rnn`: utilizando RNNs
- `05_nlp`: rocessamento de linguagem natural
- `06_transformers`: entendendo transformers
- `07_genAI_rag_agents`: entendendo IA generativa e agentes
- `08_reinforcement_and_images`: entendendo aprendizado por reforço

## Observação
Os datasets utilizados não estão incluídos neste repositório por motivos de tamanho e direitos autorais.
